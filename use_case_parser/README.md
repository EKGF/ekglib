# Use Case Parser

## Usage

```bash
python3 -m ekg_lib.use_case_parser --help
```

```text
usage: python3 -m ekg_lib.use_case_parser [-h] [--verbose] [--input INPUT]
  [--output OUTPUT]

Checks a use case RDF file

optional arguments:
  -h, --help            show this help message and exit
  --verbose, -v         verbose output
  --input INPUT, -i INPUT
                        The input .ttl file containing the definition of the
                        use case
  --output OUTPUT, -o OUTPUT
                        The output file

Currently only supports turtle.
```
